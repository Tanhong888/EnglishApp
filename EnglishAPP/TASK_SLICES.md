## 6. 规则级验收映射（Given-When-Then）

1. 同词多文章来源
- Given 用户在文章 A 已收藏单词 W
- When 用户在文章 B 再次收藏单词 W
- Then 数据层保留两条来源记录；生词列表默认按词去重展示，按文章来源筛选可分别看到 A/B

2. 收藏文章完整链路
- Given 用户未收藏文章 X
- When 用户执行收藏、重复收藏、取消收藏
- Then 收藏接口幂等；取消后可再次收藏；阅读历史不受影响

3. 列表分页稳定性
- Given 内容库有足够文章
- When 请求 `GET /articles` 的第 1 页和第 2 页（同一筛选与排序）
- Then 返回 `items/page/size/total/has_next`；两页无重复，顺序稳定

4. TTS 失败态可观测与降级
- Given 文章音频生成超过重试上限
- When 客户端请求音频状态
- Then 返回 `failed`；前端隐藏播放控件并提示“稍后重试/使用文本阅读”

5. 登出后 Refresh Token 失效
- Given 用户持有有效 Refresh Token
- When 用户调用登出后再次刷新 Token
- Then 刷新请求返回未授权
